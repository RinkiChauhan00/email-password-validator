# 📧 Email & Password Validator

A simple **Email & Password Validator** built using **HTML, CSS, and JavaScript**.

## ✨ Features

- Email format validation
- Strong password validation using Regular Expressions (Regex)
- Error messages for invalid inputs
- Success message for valid inputs
- Responsive UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📚 Concepts Practiced

- DOM Manipulation
- Form Validation
- Event Listeners
- Regular Expressions (Regex)
- `preventDefault()`
